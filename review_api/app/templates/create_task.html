Amazon
Mandatory params:
  • product_id - ASIN, it is using to request reviews from Amazon's API. For example: B00IEGN7ES
Optional params:
  • product_url - for example: https://www.amazon.com/dp/B00IEGN7ES
Walmart
Mandatory params:
  • product_id - Tool ID or any other unique id. For example: 46784935
  • product_url - mandatory if UPC or any other unique value was used for product_id param. For example: https://www.walmart.com/ip/46784935
Toysrus (Babiesrus)
Mandatory params:
  • product_id - product id from url or any other unique id. For example: 120590226
  • product_url - mandatory if product_id is not the same as id in url. For example: https://www.toysrus.com/product?productId=120590226
