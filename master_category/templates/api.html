API

Send POST requests to /converter/api to use the API.

Each POST request should contain the following values:

input_type = request.form.get('input_type', None) input_file = request.files.get('input_file', None) mapping_file = request.files.get('mapping_file', None) output_type = request.form.get('output_type', None)
  • input_type - Input file type ( available types: "amazon", "template" )
  • input_file - Input file
  • mapping_file - Additional file that will be used on conversion
  • output_type - Output file type ( available types: "walmart", "googlemanufacturer" )

Important : Make sure you're submitting the form with enctype=multipart/form-data.

Important : available conversions amazon->walmart, template->googlemanufacturer

The server will return a response after you submit a file. The response would be file if the conversion finished successfully, or the error description otherwise.

Export Google Manufacturer Center XML file

To export Google Manufacturer Center XML file POST request should contain the following values:

  • input_type - template
  • output_type - googlemanufacturer
  • caids_file - CSV file with CAIDs
  • server - server to retrieve products by CADIs (like "http://elsa4.contentanalyticsinc.com")
